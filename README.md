Q:为什么前后端都不用框架？

A:方便看，学习成本最低化，使用框架还要额外学习知识进行了解，不用框架仅靠门外汉水平就足够指挥AI了，但我个人希望急速完成毕设。

Q:为什么选择做网站作为本科毕设？

A：能很快就做出来，而且能吹应用需求，我本来想做miniOS、miniComplier、FPGA-Based miniCPU，但是吹需求的时候不好吹，所以选择做网站。

后端运行环境为phpStudy（nginx），数据库版本为5.5，运行浏览器为edge（147.0.3901.1）

三个角色：个人用户、企业用户、管理员用户。
